## üöÄ Features

* **Model:** Uses Google's latest `gemini-3-flash-preview`.
* **Backend:** Lightweight Flask server.
* **Frontend:** Modern Dark Mode UI using **Tailwind CSS**.
* **Containerized:** Includes a `Containerfile` for easy deployment via Podman or Docker.
* **Secure:** API Keys are passed via Environment Variables (No hardcoding).
